# 🔑 City Key Maker — Professional Locksmith & Key Services

A modern, single-page website for **City Key Maker**, a local locksmith shop in Defence Colony, Gurugram, Haryana.

🌐 **Live Site**: [citykeymaker.com](https://zyvrix.github.io/CityKeyMaker/) *(GitHub Pages)*

## ✨ Features

- **Industrial-Noir Design** — Dark steel (#0f1117) + gold/amber (#d4a017) theme
- **Animated Hero** — Canvas particle effect with shimmer sweep
- **7 Service Cards** — Key duplication, car keys, emergency unlock, lock repair & more
- **Gallery Section** — Real shop photos with hover zoom & full-screen lightbox
- **3-Step Process** — Visual stepper showing how to get service
- **Customer Reviews** — Testimonial cards with 5-star ratings
- **Contact Section** — Phone, WhatsApp, address, working hours + Google Maps embed
- **Floating WhatsApp Button** — Always-visible CTA with pulse animation
- **Fully Responsive** — Mobile hamburger menu, adaptive layouts
- **SEO Optimized** — Meta tags, semantic HTML, Open Graph tags
- **SVG Dividers** — Key & lock-shaped decorative section separators
